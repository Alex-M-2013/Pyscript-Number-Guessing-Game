# Pyscript Number Guessing Game

A simple unit converter made in HTML, CSS and Python3 (Using [pyscript](https://pyscript.net/))

__Note:__ If you want to run the code yourself you will need to start a local server (You can't just double-click the HTML file). 
For VS Code I recommend using the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension.
